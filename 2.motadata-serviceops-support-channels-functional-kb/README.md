# Motadata ServiceOps — Support Channels: Functional KB

A complete, plain-English functional guide to the **Support Channels** admin section of
Motadata ServiceOps — every page, tab, button, field, and option, in the product's own
words. Captured live from the PMG ServiceOps environment at `http://172.16.15.185` on
20 July 2026. It describes what the product is and does; it contains no opinions,
comparisons, or improvement ideas.

Support Channels covers how people reach the service desk: email in and out, the
self-service Support Portal, live Chat, and the Virtual Agent chatbot with its
per-channel messenger connections (Slack, Teams, Google Chat, Telegram, Facebook
Messenger, WhatsApp, Viber, LINE).

Start with the [overview](docs/00-overview.md), then use the index below.
