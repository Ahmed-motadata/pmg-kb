# Virtual Agent — Slack App Config

**Path:** Admin → Support Channels → Virtual Agent → Slack App Config
(`/admin/support-channels/ai-bot?tab=slack_app_config`)

Connects the Virtual Agent to Slack. The page description reads: "Connect the bot to
Slack for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Domain Name | Yes | Text | The Slack workspace domain |
| Client Id | Yes | Text | From the Slack app |
| Client Secret | Yes | Text | From the Slack app |
| Sharable URL | No | Text | |
