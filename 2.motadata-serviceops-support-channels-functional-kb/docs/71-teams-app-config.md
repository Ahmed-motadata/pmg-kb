# Virtual Agent — Teams App Config

**Path:** Admin → Support Channels → Virtual Agent → Teams App Config
(`/admin/support-channels/ai-bot?tab=teams_app_config`)

Connects the Virtual Agent to Microsoft Teams. The page description reads: "Connect the
bot to Microsoft Teams for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Allow Notification | — | Toggle (Off) | Let the bot push notifications to Teams |
| Client Id | Yes | Text | From the Teams app registration |
| Password | Yes | Password (eye reveal) | |
| Attachments | — | **Attach Files** upload button | |
