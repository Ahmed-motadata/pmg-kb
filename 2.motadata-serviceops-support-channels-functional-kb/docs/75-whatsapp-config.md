# Virtual Agent — WhatsApp Config

**Path:** Admin → Support Channels → Virtual Agent → WhatsApp Config
(`/admin/support-channels/ai-bot?tab=whatsapp_config`)

Connects the Virtual Agent to WhatsApp. The page description reads: "Connect the bot to
WhatsApp for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Allow Notification | — | Toggle (Off) | |
| Client Id | Yes | Text | |
| Phone Number ID | Yes | Number (default 0) | The WhatsApp Business phone number id |
| Access ID | Yes | Text | |
| Allow Guest User | — | Toggle (Off) | |
