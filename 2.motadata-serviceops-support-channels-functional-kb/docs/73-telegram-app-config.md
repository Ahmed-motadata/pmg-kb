# Virtual Agent — Telegram App Config

**Path:** Admin → Support Channels → Virtual Agent → Telegram App Config
(`/admin/support-channels/ai-bot?tab=telegram_app_config`)

Connects the Virtual Agent to Telegram. The page description reads: "Connect the bot to
Telegram for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Client Id | Yes | Text | The Telegram bot token/id |
| Allow Guest User | — | Toggle (Off) | Let users who are not registered requesters talk to the bot |
