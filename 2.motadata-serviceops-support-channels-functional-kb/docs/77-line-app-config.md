# Virtual Agent — Line App Config

**Path:** Admin → Support Channels → Virtual Agent → Line App Config
(`/admin/support-channels/ai-bot?tab=line_app_config`)

Connects the Virtual Agent to LINE. The page description reads: "Connect the bot to
LINE for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Client Id | Yes | Text | |
| Allow Guest User | — | Toggle (Off) | |
