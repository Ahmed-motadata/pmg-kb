# Virtual Agent — Viber App Config

**Path:** Admin → Support Channels → Virtual Agent → Viber App Config
(`/admin/support-channels/ai-bot?tab=viber_config`)

Connects the Virtual Agent to Viber. The page description reads: "Connect the bot to
Viber for chat-based support." **Update** saves; **Cancel** discards.
