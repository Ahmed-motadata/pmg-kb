# Chat

**Path:** Admin → Support Channels → Chat (`/admin/support-channels/chat`)

Turns on live chat between requesters and technicians and configures who answers it and
how it looks. The page description reads: "Enable live chat for real-time technician
support." **Update** saves; **Cancel** discards.

## Settings

Top toggles (state shown is this environment's):

| Setting | State here |
|---|---|
| Allow Chat Support | On |
| Allow Collaboration Support | Off |
| Allow Requester to view My Chats | On |

## Support Engineers section

| Field | Type | Notes |
|---|---|---|
| Technician | Multi-pick search | Searches technicians; picked people answer chats. Here: Stuti (stuti.sanghvi@motadata.com) |
| Technician Group | Search picker | Searches technician groups |
| Display First Name Only | Toggle (Off here) | Show only first names to requesters |

## Messages and appearance

| Field | Required | Value here |
|---|---|---|
| Welcome Message | Yes | "Hi! Welcome to Chat, One of our technicians will be with you soon." |
| Missed Message | Yes | "All our technicians are busy at the moment, please try later." |
| Font Family | No | A choice control (left unset here) |
| Timeout | Yes | 60, with a "Second" unit label |
| Chat Logo (50px X 50px for better resolution ) | — | **Upload Chat Logo** button, with an info icon |
| Chat Favicon (30px X 30px For Better Resolution) | — | **Upload Chat Favicon** button, with an info icon |
