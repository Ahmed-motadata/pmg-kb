# Virtual Agent — Google Chat App Config

**Path:** Admin → Support Channels → Virtual Agent → Google Chat App Config
(`/admin/support-channels/ai-bot?tab=google_chat_app_config`)

Connects the Virtual Agent to Google Chat. The page description reads: "Connect the bot
to Google Chat for chat-based support." **Update** saves; **Cancel** discards.

| Field | Required | Type | Notes |
|---|---|---|---|
| Enabled | — | Toggle (Off here) | When On, the fields below appear |
| Allow Notification | — | Toggle (Off) | |
| Service Account Details | Yes | Text | The Google service-account credentials |
