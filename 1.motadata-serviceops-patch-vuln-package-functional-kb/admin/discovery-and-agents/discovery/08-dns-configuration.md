# Admin — DNS Configuration

## Purpose
**Set the DNS servers used to resolve discovered hosts** during discovery scans (maps IPs to hostnames).

## Access / Navigation
Admin → **Discovery And Agents → Discovery → DNS Configuration**. URL: `/admin/asset-management/discovery?tab=dns_configuration`.

## Settings
- **Primary DNS** — server address
- **Secondary DNS** — server address
- Edit via the pencil (✎) icon.
