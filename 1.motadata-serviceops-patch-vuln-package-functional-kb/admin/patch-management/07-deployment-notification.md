# Admin — Deployment Notification

## Purpose
Notify users and technicians about patch deployments — **set up alerts for patch deployment status** on a recurring schedule.

## Access / Navigation
Admin → **Patch Management → Patch Administration → Deployment Notification**. URL: `/admin/patch-management/patch-settings?tab=deployment_notification`.

## Settings
- **Deployment Notification Frequency** — number input + unit **Hours** (e.g. every 1 hour).
- Shows **Next Execution Time** (e.g. "Sun, Jul 19, 2026 11:00 PM").
- Actions: **Update** / **Cancel**.
