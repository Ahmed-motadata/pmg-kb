# Motadata ServiceOps — Task Management: Functional KB

A complete, plain-English functional guide to the **Task Management** admin section of
Motadata ServiceOps — every page, tab, button, field, and option, in the product's own
words. Captured live from the PMG ServiceOps environment at `http://172.16.15.185` on
21 July 2026. It describes what the product is and does; it contains no opinions,
comparisons, or improvement ideas.

Task Management configures how tasks work across the product: their types, the fields on
the task form, the rules that show/hide/require those fields, and the statuses a task
moves through.

Start with the [overview](docs/00-overview.md), then use the index below.
